AIM

Provide an OpenCPN Autopilot Control for Raymarine Autopilots. Autopilot can control a Raymarine autopilot by sending and receiving special NMEA messages to the course-computer.

Beta Version

by Bernd Cirotzki

“Autopilot can control a Raymarine autopilot by sending and receiving special NMEA messages to the course-computer.”

“For this functioning it is necessary to convert a messages from “NMEA” to “SEATALK”. For this conversion a special hardware is required. (Seatalk – NMEA converter, Seatalk-link) The hardware can be ordered here : http://www.gadgetPool.de

“It is tested with a Raymarine Smartpilot S1G an a ST6002+ Panel. But I think it would work with others too.”

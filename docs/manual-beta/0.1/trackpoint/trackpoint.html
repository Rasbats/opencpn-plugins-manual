Trackpoint

Aim

Plotting ORE 4410D-01 TRACKPOINT II Plus SYSTEM** target data on OpenCPN.

This plugin translates NMEA sentences from ORE Trackpoint II to AIS targets and NMEA Depth. Translated data are emmited as NMEA sentences so they can be used by other plugins.
